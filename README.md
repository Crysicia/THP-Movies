# THP-Movies !

## :wrench: Installation guide
To install and test our awesome project :

    $ git clone https://github.com/Crysicia/THP-Movies.git
    $ cd THP-Movie
    $ bundle install
    $ rails db:create
  Run it like this :

    $ rails s
    
  Then you can go to :

    $ localhost:3000
  And run the tests :
  
    $ rails test:system
    
## :radio: What does it do ?
Simple-yet-effective TMDb scrapper and display.
